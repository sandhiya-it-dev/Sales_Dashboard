# 📊 Sales Dashboard

## Project Overview

Sales Dashboard is a Streamlit-based web application developed using Python, PostgreSQL, SQLAlchemy, Pandas, and python-dotenv. The application helps businesses manage sales, payments, branch performance, and business analytics through an interactive dashboard.

## Technologies Used
- Python
- Streamlit
- PostgreSQL
- SQLAlchemy
- Pandas
- python-dotenv

## Features

### 🔐 User Authentication
- Secure login system
- Role-based access control
- Super Admin and Branch User access

### 📊 KPI Dashboard
- Total Sales
- Total Received Amount
- Total Pending Amount
- Pending Percentage

### ➕ Sales Management
- Add new sales records
- View sales reports
- Track sales status

### 💰 Payment Management
- Add customer payments
- View payment details
- Payment method analysis

### 📈 Analytics
- Business Analytics
- Open vs Close Sales Analysis
- Branch Performance Analysis
- Sales Trend Analysis

### 🧠 SQL Reports
- Top Sales
- Open Sales
- Closed Sales
- Pending Payments

## Database Tables
- branches
- users
- sales
- payments
